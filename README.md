# 🏋️ Gym Management System (Java SE - CLI)
Sistema de gerenciamento de academia desenvolvido em Java SE, operado via terminal (CLI), com foco em Programação Orientada a Objetos (POO) e uso de Collections.

# 📌 Objetivo
Aplicação para gestão de alunos e planos de uma academia, permitindo controle de dados em memória através de um CRUD completo, filtros de busca e geração de relatórios.

# 🛠 Tecnologias Utilizadas
- Java SE
- Programação Orientada a Objetos (POO)
- Collections Framework (List, ArrayList)
- Execução via Terminal (CLI)

# 📚 Conceitos Aplicados
! ✅ Encapsulamento

 - Atributos privados.
 -  Uso de getters e setters.

✅ Abstração e Herança

- Plano é uma classe abstrata.
- Especializações de plano implementam comportamentos específicos.

✅ Polimorfismo

- O método calcularMensalidade() possui comportamentos diferentes dependendo do tipo de plano.

✅ Collections

- Uso de ArrayList<Aluno> para armazenamento dinâmico.
- Proibido uso de arrays fixos.

# 🚀 Funcionalidades
### 📌 CRUD Completo de Alunos
- ✔ Cadastrar aluno
- ✔ Listar todos os alunos
- ✔ Atualizar dados de um aluno
- ✔ Remover aluno
- ✔ Buscar aluno por ID

### 🔍 Filtros de Busca
- Buscar alunos por nome
